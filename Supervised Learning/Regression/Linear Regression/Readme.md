# 📊 Salary Prediction using Linear Regression

## 📌 Overview

This project predicts salary based on **years of experience** using **Linear Regression**.

---

## 📂 Dataset

* Salary Data
* Features:

  * Years of Experience (Input)
  * Salary (Target)

---

## ⚙️ Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* Matplotlib

---

## 🔄 Workflow

1. Load Data
2. Explore & Clean
3. Train-Test Split
4. Train Model
5. Predict
6. Evaluate & Visualize
