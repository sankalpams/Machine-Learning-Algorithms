# 🐦 Twitter Sentiment Analysis using Naive Bayes

A professional Natural Language Processing (NLP) project that performs Twitter sentiment classification using the Naive Bayes algorithm.

This project uses:
- Text preprocessing
- NLP feature extraction
- CountVectorizer
- Multinomial Naive Bayes
- Data visualization
- Model evaluation metrics

The goal is to classify tweets as:
- Positive
- Negative

---

# 📌 Project Overview

This project demonstrates how Machine Learning and Natural Language Processing (NLP) can be used to analyze sentiment from Twitter data.

The model was trained using:
- Twitter sentiment dataset
- Binary sentiment labels
- Multinomial Naive Bayes classifier

The project also includes professional visualizations to analyze:
- Sentiment distribution
- Model performance
- Word frequency
- Classification accuracy
- Tweet sentiment patterns
