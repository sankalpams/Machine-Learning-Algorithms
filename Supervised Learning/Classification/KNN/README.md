# 🌸 Iris Flower Classification using K-Nearest Neighbors (KNN)

## 📌 Project Overview
This project implements the K-Nearest Neighbors (KNN) algorithm, a supervised machine learning technique used for classification problems.

The model analyzes the Iris flower dataset and classifies flowers into different species based on their sepal and petal measurements.

---

## 🎯 Objective
The main objective of this project is to:
- Understand supervised learning
- Implement the K-Nearest Neighbors algorithm
- Perform feature scaling
- Visualize data distributions
- Evaluate classification performance

---

## 📊 Dataset Information
Dataset used:

## 🌐 Iris Flower Dataset

### Dataset Features

| Feature | Description |
|---|---|
| SepalLengthCm | Sepal length in centimeters |
| SepalWidthCm | Sepal width in centimeters |
| PetalLengthCm | Petal length in centimeters |
| PetalWidthCm | Petal width in centimeters |
| Species | Iris flower species |
